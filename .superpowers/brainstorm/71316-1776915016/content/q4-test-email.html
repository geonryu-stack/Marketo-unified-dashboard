질문 4 — 테스트 메일은 언제 확인하나요?

에셋 + URL 조합을 확인하는 테스트 메일 타이밍을 선택해 주세요.

A

예약 전 한 번만 확인 (추천)

에셋 + URL을 입력한 뒤 "테스트 발송" 버튼 클릭 → 메일 확인 → 이상 없으면 그룹별 일정 설정하고 전체 예약.

어차피 같은 에셋+URL이 모든 그룹에 들어가므로 한 번만 확인해도 충분

B

예약 직전 자동 테스트 (현재 방식)

각 그룹을 예약할 때 Phase 1이 자동으로 테스트 메일을 발송. 승인 이메일을 확인 후 개별 승인.

그룹마다 승인이 필요해 여러 번 확인해야 함
